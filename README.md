# app-aus10-notes

## 项目简介

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面通常用于记录、测试或展示某些功能、样式或交互效果，不针对任何具体网站或域名。仓库内容以静态 HTML 文件为主，可直接通过 GitHub Pages 或其他静态托管服务访问。

## 目录说明

- `/`：根目录下存放各独立 HTML 页面文件，每个文件通常为一个完整的页面或功能演示。
- `/assets/`（如有）：存放页面所需的样式、脚本、图片等公共资源。
- `/docs/`（如有）：存放项目相关文档或说明文件。

## 页面归档说明

- 每个 HTML 页面均为独立、自包含的文件，尽量不依赖外部资源（除公共库外）。
